# 🏦 AI Credit Risk Lending Decision Engine

## 📌 Project Overview

The **AI Credit Risk Lending Decision Engine** is an end-to-end Machine Learning application that predicts the probability of loan default and assists lenders in making informed lending decisions.

The project combines data preprocessing, feature engineering, machine learning, explainable AI concepts, and an interactive Streamlit web application to evaluate borrower risk based on financial information.

The application estimates the probability of default, categorizes borrowers into different risk levels, and provides lending recommendations along with a business-friendly explanation.

---

# 🚀 Features

* Predicts probability of loan default using Machine Learning
* Interactive Streamlit web application
* Automatic monthly EMI calculation
* Borrower risk categorization
* Lending decision recommendation
* Business-friendly decision explanation
* Key risk factor identification
* Professional dashboard interface

# ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project directory

```bash
cd AI-Credit-Risk-Lending-Decision-Engine
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Application

```bash
streamlit run app/app.py
```
